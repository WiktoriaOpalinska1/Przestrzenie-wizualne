# Przestrzenie-wizualne

Projekt grupowy w ramach zajęć z Technologii Kognitywnych na UJ.
Grupa realizowała temat: Przestrzenie Wizualne. 

Celem projektu było stworzenie interaktywnego systemu, który na podstawie wykrywania markerów AR (hiro, kanji) prezentuje wybrane efekty wizualne odzwierciedlające stan emocjonalny. Pełen opis projektu wraz z podziałem obowiązków znajduje się w pliku pdf.
